# PeaversConfig

A centralized configuration hub for all Peavers addons. Provides a unified settings window where you can configure every Peavers addon from one place.

**Website:** [peavers.io](https://peavers.io) | **Addon Backup:** [vault.peavers.io](https://vault.peavers.io) | **Issues:** [GitHub](https://github.com/peavers-warcraft/PeaversConfig/issues)

## Features

- Single configuration window for all Peavers addons
- Per-addon settings pages with tabbed navigation
- Global appearance settings applied across all addons
- Profile management with ecosystem-wide presets
- Changelog viewer for all installed Peavers addons

## Installation

1. Download from [CurseForge](https://www.curseforge.com/wow/addons/peaversconfig)
2. Ensure [PeaversCommons](https://www.curseforge.com/wow/addons/peaverscommons) is also installed
3. Enable the addon on the character selection screen

## Usage

- `/pconfig` - Open the configuration window
- `/peavers` - Open the configuration window
- `/peavers <addon>` - Jump directly to a specific addon's settings

## Dependencies

- [PeaversCommons](https://www.curseforge.com/wow/addons/peaverscommons) (required)
